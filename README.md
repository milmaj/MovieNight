# Movie Night

[<img src="https://github.com/milmaj/MovieNight/blob/master/src/assets/screenshot-2.png">](https://milmaj.github.io/movie-night/)

## Features

- Search
- Movie Details
- Movie Trailers

## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```

Build for production,

```bash
$ yarn build
```
