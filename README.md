# Movie Night

![Details Page](https://github.com/milmaj/MovieNight/blob/master/src/assets/screenshot-2.png)

## Features

- Search
- Movie Details
- Movie Trailers

## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```

Build for production,

```bash
$ yarn build
```
